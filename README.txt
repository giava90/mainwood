Important steps before running the script convert_data.py
1) make sure that the folders paths are corrects.. This also requires to create a specific structure for saving the outputs.. For each <CaseStudyRegion> and <ManagementScenario> 
1.1) mkdir <CaseStudyRegion>
1.2) mkdir <CaseStudyRegion>/intermediate <CaseStudyRegion>/outputs
1.3) mkdir <CaseStudyRegion>/intermediate/<ManagementScenario> <CaseStudyRegion>/<ManagementScenario>
2) to run the script you need to change their mode:
chmod u+x run_conversion.sh
chmod u+x run_analysis.sh
