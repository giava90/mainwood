Important steps before running the script convert_data.py
1) make sure that the folders paths are corrects
2) make sure that the file templateSpec_v2.txt is present in all folders 
3) to run the script
python convert_data.py "BIO"  "True" 2 

where "BIO" is an example of forest management, "True" or "False" for using a sample of the data, 2 is the number of cores used
